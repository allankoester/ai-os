# Personal Knowledge (private)

Everything in this folder is **private to this machine and this user**.

- Excluded from git via `.gitignore` (only this README and `.gitkeep` are tracked).
- Never synced to OneDrive company shares, never referenced in company artifacts.
- Agents may read from here for your own context, but must not copy personal
  content into `knowledge/company/` without your explicit instruction.

Suggested structure (free-form, Markdown):

```
personal/
├── notes/          personal working notes
├── contacts/       personal relationship notes
├── drafts/         private drafts not ready for company knowledge
└── memory.md        optional personal agent memory (private)
```

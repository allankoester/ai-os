# Inbox (unsorted intake)

Drop new, unclassified material here: meeting notes, exports, ideas, client
input, screenshots-as-markdown, anything.

Rules:

1. Nothing in `inbox/` is a source of truth. Agents must not cite inbox files
   in final outputs.
2. Classification is Mara's job (`knowledge_intake_workflow`): Mara proposes a
   target (`company/<domain>/` or `personal/`), the user confirms, the file is
   moved, and the inbox copy is deleted.
3. Keep the inbox small. If a file sits here longer than two weeks, classify or
   delete it.

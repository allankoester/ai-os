## Hinweise für Noah

- **Muster übertragen**, nicht direkt kopieren — Stil, Struktur und Sprache aus erfolgreichen Prompts ableiten
- **Negativen Prompt immer anpassen** — generische Begriffe aus der Library sind Ausgangspunkt, nicht Endpunkt
- **Neue erfolgreiche Prompts** können vom User hier ergänzt werden
- **Leere Kategorien** ignorieren — Library wächst nach Bedarf

# Quality Rubric for External Outputs (Stage 1 gate)

Score each dimension Pass / Revise. Any Revise blocks approval.

| # | Dimension | Pass means | Score |
| --- | --- | --- | --- |
| 1 | Grounding | Every factual claim traces to a `knowledge/company/` source or explicit user input | |
| 2 | Strategic fit | Matches Steadymade positioning, ICP and market scope; Strategy Gate passed where required | |
| 3 | Voice | Clear, grounded, operational, calm; no hype, no filler, no "not A, but B" pattern | |
| 4 | Claims discipline | No inflated ROI promises, no fake urgency, no one-size-fits-all claims | |
| 5 | Clarity | A busy mid-market decision maker understands the point on first read | |
| 6 | Completeness | Format, length and structure match the brief's Required Output | |
| 7 | Data hygiene | No personal knowledge, no inbox content, no confidential client data | |

## Usage

- Rosa applies this rubric during editorial review.
- Danny confirms the rubric result before requesting user approval.
- Record the result in the run log (`runs/run-log-template.md`).
